Useful Links

This page contains a library of links to useful resources for teachers and students of Computer Science in Ireland. Each link has a short description and is organised by topic. The date in brackets is when the link was last checked.

Links

Below are a series of curated links broken down by topic.

General Purposes

  • Teach with ICT – Good website with strategies and resources for teaching ICT (02/2025)

IDEs Offline

  • Thonny – Basic IDE for working with Python
  • Visual Studio Code – More powerful IDE from Microsoft
  • Github Desktop – Clone Git repositories from GitHub to a local folder

IDEs Online

  • Replit – Online IDE for multiple languages; limits on free account
  • Google CoLab – Jupyter Notebook (Python) editor; allows multiple programs in one notebook
  • Trinket.io – Online IDE that can be embedded in other pages; free version limited to Python 2
  • VSCode Dev – Online Visual Studio Code; access GitHub repos and local folders; many of the same extensions
  • Codepad – Similar to Replit and Trinket; more flexible than Trinket (free for now)

Computer Science

  • Teckno CS Course – Free Computer Science and coding courses from Dell for Leaving Cert CS

Python Coding

  • Turing Labs – Learn Python from Amazon Future Engineer
  • Python CheatSheet – Python cheat sheet

Networking

  • Network Animations – Network animations explaining how they work
  • Network Protocol Matching Quiz – Word matching quiz for network protocols
  • Network Hardware Word Match Quiz – Word match quiz on network hardware
  • Network Design Drag and Drop – Identify common network hardware

AI Resources

  • ChatGPT Cheatsheet – Help creating prompts for ChatGPT and other LLMs

Data Sources (ALT2)

  • Our World in Data – Free data sources for analytics
  • Statista – Free-to-use data sources
  • Kaggle – Free datasets; no login required

Operating Systems

  • OS Definitions – Match OS words to their definitions
  • Virtual Machines Online – Run Windows, Mac or Linux in a VM in the browser
  • PCJs machines – Old operating systems running in browser emulators
  • Web OS – Run versions of operating systems including Mac OS online
  • Operating System Library – Disk images of old OSes for virtual machines

Micro bit

  • Microbit and AI – Explore AI through movement and machine learning (ML)

Logic Gates

  • Logicly – Create and simulate logic gates online (free trial)
  • Logic Circuit – Free software for logic circuits and truth tables
  • Everycircuit – Online electronic circuit simulator including logic gates
